Detailed Description

Messenger class that defines commands for TG4RunManager.

Implements commands:

  • /mcControl/root
  • /mcControl/rootMacro [macroName]
  • /mcControl/rootCmd [cmdString]
  • /mcControl/useRootRandom [true|false]
  • /mcControl/g3Defaults
