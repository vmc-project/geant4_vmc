Detailed Description

Implementation of the GarfieldG4FastSimulationModel class.

Garfield garfieldpp example adapted to Virtual Monte Carlo. This class is imported from garfieldpp example. There is disabled WriteGeometryToGDML function in order to avoid requiring Geant4 optional library.
